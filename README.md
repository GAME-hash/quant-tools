# 量化工具箱 · QuantTools

A股投资者的 AI 辅助决策工具集。5 个工具，零服务器，纯前端。

## 工具列表

| 工具 | 描述 | 价格 |
|------|------|------|
| 🌡️ 大盘温度计 | 市场情绪量化，一个数字看懂冷热 | 免费 |
| 🩺 持仓体检单 | 持仓诊断：集中度、行业分布、健康评分 | ¥3.9 |
| 💰 成本精算器 | A股交易成本精确计算 | ¥3.9 |
| 🛑 止损线计算器 | 保守/中性/激进三方案对比 | ¥3.9 |
| 📊 PE/PB 分位器 | 估值历史分位可视化 | ¥3.9 |

## 技术

- 零服务器：所有工具均为静态 HTML/JS
- 数据来源：新浪财经 + 东方财富 + akshare
- 暗色主题，移动端优先
- 由 [Hermes Agent](https://github.com/NousResearch/hermes-agent) 驱动开发
